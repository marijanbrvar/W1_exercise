Set up a GitHub repository with correct tools

In this exercise, we set up a "Hello world" repository. No complex coding is required for this exercise. Our goal here is to master all tools and best practices you learned about in previous steps. I will be using them in all Microverse project and most likely in your future job as well!

Task:

  1. Create new Github repo.
  2. Create a new branch.
  3. Set up linter for HTML&CSS.
  4. Add 3 files:
    1. A .html file that includes a header with text "Hello world!"
    2. A .css file that includes one class with styles for your header (make it in your favourite color)
    3. A test.md file with text "This file should be ignored by git"
  5. Make sure that the last file is really ignored by git.
  6. Add README file to your project.
  7. Add each file in a separate commit.
  8. Open a Pull Request and ask your coding partner for a code review.
